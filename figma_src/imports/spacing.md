## Border radius (4 values)

| Token | CSS var | Value | Usage |
|-------|---------|-------|-------|
| scale0 | --border-radius-0 | 0px | Square edges (tables, dividers) |
| scale200 | --border-radius-200 | 8px | Buttons, inputs, chips, stickers, cards |
| scale400 | --border-radius-400 | 16px | Nav buttons, suggestion cards, modals |
| scale700 | --border-radius-700 | 28px | Profile avatars, pills |

## Border width (4 values)

| Token | CSS var | Value | Usage |
|-------|---------|-------|-------|
| scale0 | --border-width-0 | 0px | No border |
| scale25 | --border-width-25 | 1px | Default borders (cards, inputs, dividers) |
| scale50 | --border-width-50 | 2px | Focused inputs, active states |
| scale100 | --border-width-100 | 4px | Profile avatar border |

## Spacing rules

- **Gap between nav buttons**: 10px
- **App bar padding**: 16px vertical, 24px horizontal
- **App bar internal gap**: 8px (category) or 24px (fiche)
- **List item padding**: 12px vertical, 24px horizontal
- **Card padding**: 16px all sides
- **Section gap**: 24px between sections
- **Page margins**: 24px (desktop), 16px (mobile)
