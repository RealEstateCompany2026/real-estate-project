# AI Components

All AI-related UI elements use branded (purple) as their accent color to signal "generated by AI".

## Atome: AI Suggestion Badge
- **Usage**: Counter badge showing number of AI suggestions available
- **Size**: 33×24px min
- **Radius**: scale400 (16px)
- **Border**: scale25 (1px), branded/500
- **Surface**: branded/50 (light) / branded/800 (dark)
- **Text**: 12px/600, neutral/500 (light) / neutral/200 (dark)
- **Icon**: branded/500
- **States**: "none" (hidden when count=0), "some" (shows count: 1, 3, 12, 99)

## Dashboard: Suggestions Panel
- **Usage**: Dashboard widget showing AI suggestion categories
- **Surface**: neutral/white (light) / neutral/800 (dark)
- **Border**: neutral/50 (light) / neutral/600 (dark), scale25, radius scale400
- **Categories**: Conseil, Service, Administratif, Transaction
- **Action**: "Voir les suggestions" branded button

## Organisme: Suggestion Card
- **Usage**: Individual AI suggestion with action button
- **Surface**: neutral/white (light) / neutral/800 (dark)
- **Border**: neutral/50 (light) / neutral/600 (dark), scale25, radius scale400
- **Content**:
  - Title (paragraph-md semibold): suggestion text, e.g., "Contacter le client pour relance"
  - Subtitle (paragraph-sm regular, neutral/400): context, e.g., "Inactivité détectée depuis 30 jours"
  - Action button: "Appliquer" (branded variant)
- **Icon**: branded/500 (IA sparkle icon)

## Usage rules
- Any content generated by Claude API must be marked with the AI suggestion badge or displayed in an AI suggestion card
- AI-generated text (descriptions, emails, scripts) should show a small "✨ IA" badge next to them
- User can always edit AI-generated content — show "ai_generated" or "ai_edited" source tag
