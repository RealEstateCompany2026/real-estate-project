# Typography — Design Token Guidelines

## Font family
Always use **Roboto** for all text.

## Headings — Bold 700

| Level | Desktop size/lh | Mobile size/lh |
|-------|----------------|----------------|
| H1 | 48px / 58px | 32px / 40px |
| H2 | 40px / 48px | 28px / 32px |
| H3 | 32px / 38px | 26px / 30px |
| H4 | 28px / 34px | 24px / 28px |
| H5 | 24px / 28px | 22px / 26px |
| H6 | 20px / 24px | 20px / 24px |

## Paragraphs — 3 weights (Light 300 · Regular 400 · Semibold 600)

| Size | Desktop size/lh | Mobile size/lh |
|------|----------------|----------------|
| paragraph-lg | 20px / 24px | 20px / 24px |
| paragraph-md | 16px / 20px | 16px / 20px |
| paragraph-sm | 14px / 16px | 14px / 16px |
| paragraph-xsm | 12px / 14px | 12px / 14px |

## Usage rules

- **Page titles**: Always H4 Bold (28px), color `--text-headings` (neutral/600)
- **Section headings**: H5 or H6 Bold
- **Body text**: Always paragraph-md Regular (16px/400), color `--text-body`
- **Table headers**: paragraph-xsm Semibold (12px/600), color neutral/300, uppercase, letter-spacing 0.5px
- **Captions and labels**: paragraph-sm Regular (14px/400), color `--text-caption`
- **Small text**: Only use paragraph-xsm for timestamps, counters, and metadata — never for readable body content
- **Sticker text**: 12px/600 Semibold, uppercase, letter-spacing 0.3px
- **Button text**: paragraph-md Semibold (16px/600) for standard buttons, paragraph-sm Semibold (14px/600) for compact buttons
